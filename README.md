# TrustGrid

Autonomous, continuous vendor risk and compliance orchestration (Track 2: Active Agents).

## How to run

```bash
pnpm install
pnpm test          # unit + integration
pnpm run test:e2e  # Playwright E2E (starts Next.js)
pnpm --filter web dev   # dev server at http://localhost:3000
```
